## 面试官为何靠算法
   - 算法 考察面试者思维能力和解决问题的能力，而非单纯的编码能力
   - 抽象能力（数据结构+算法），是否思路清晰
     - 数组、字符串、栈、队列、链表、哈希表、二叉树（堆）
     - 递归（程序策略）、单双指针、dummy 节点、滑动窗口、排序（快排）、查找、分治（二分）、回溯、动态规划、贪心算法
   - 有3道，半道很难，看是否抉择并且当场学习的能力，未来解决复杂问题的能力（师徒关系）
   - 算法可以提升效率，时空复杂度
   - 优秀程序员是有优秀算法的能力
